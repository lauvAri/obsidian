## 动态绑定机制
```
Object o = new GeometricObject();
System.out.println(o.toString());
```
- o的声明类型为object
- o的实际类型为GeometricObject
- o调用的是哪个toString()是由实际类型决定的
## 方法匹配和绑定
- 匹配和绑定是独立过程
- 编译时只会按照声明类型去匹配方法
